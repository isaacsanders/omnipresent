# Omnipresent

Omnipresent is a gem that is meant to be a framework for building/writing apps everywhere.

## Installation

Add this line to your application's Gemfile:

    gem 'omnipresent'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omnipresent
